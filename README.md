# full_stack_development_project4

## Project Goal

The aim of this project is to build a full-stack task management system that allows teams to create projects, assign tasks, track progress, and manage work securely through role-based access. 

## Project Structure

### Epics

#### Epic 1 - Project Management

##### User stories

As a project manager, I want to create a new project so that I can organise my work.

As a project manager, I want to view and project details so that I can understand the status of a project.

As a project manager, I want to edit project details so that project information stays accurate. 

As a programme manager, I want to view a list of projects so that I can see all projects in my portfolio. 

As a programme manager, I want to delete projects so that closed or cancelled projects can be removed. 

As a team member, I want to see projects that I am assigned to so that my responsibilities are clear.
